# SchScie

A high school science class emulation animations demo site.

## Overview

SchScie is a public demo site featuring interactive animations designed to help high school students visualize and understand scientific concepts. The project aims to make science education more engaging through visual demonstrations.

## Features

- Interactive science animations
- Educational demonstrations for high school curriculum
- Easy-to-understand visualizations of scientific concepts

## Getting Started

Visit the demo site to explore the available science animations.
